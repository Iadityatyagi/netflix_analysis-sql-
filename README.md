## Netflix SQL Analysis Project

### Project Overview

This project focuses on analyzing Netflix Movies and TV Shows data using SQL in PostgreSQL/MySQL Workbench. I solved 15 business-focused SQL questions covering data exploration, filtering, aggregations, subqueries, CTEs, and window functions to extract meaningful insights from the Netflix dataset. 

### Skills Used

* SQL
* PostgreSQL / MySQL Workbench
* Data Analysis
* Aggregations
* Joins
* Subqueries
* CTEs
* Window Functions
* Data Filtering & Sorting

### Key Analysis Performed

* Counted total Movies and TV Shows
* Identified top content-producing countries
* Analyzed yearly content releases
* Found most common content ratings
* Filtered TV Shows based on seasons
* Analyzed Indian movies by genre
* Identified top directors by movie count
* Found years with highest content additions
* Retrieved oldest and latest movies
* Used Window Functions and Subqueries for advanced analysis 

### Project Outcome

Performed end-to-end SQL analysis on Netflix data to improve data querying, analytical thinking, and database management skills while solving real-world business questions.
